# 摸鱼补给站

一个纯前端摸鱼小工具，直接打开 `index.html` 就能玩。

## 操作

- 拖动小鱼吃气泡，获得摸鱼值和连击。
- 大鲨鱼会持续追逐小鱼，被追上后本局结束。
- 撞到灰色障碍物会短暂减速，并清空连击。
- 吃到气泡可以把鲨鱼稍微甩开一点。
- 使用“鲨鱼速度”滑轨调节追击速度，当前局立即生效。
- 使用“鲨鱼数量”滑轨调节追击数量，可选 `0-3` 条；`0` 条时是休闲模式。
- 按 `Space` 暂停或继续。
- 按 `Esc` 快速隐藏，页面会切换成工作流看板。
- 按 `Alt + H` 也可以快速切换到伪装页面。
- 在工作流看板点击右上角返回按钮继续，游戏结束后可以重新开始。
- 点击右上角 `◎` 开启老板雷达，授权摄像头后会在本地检测人脸。
- 老板雷达检测到摄像头里出现 2 张及以上人脸时，会自动切换到工作流看板。

## 老板雷达说明

- 摄像头画面只在浏览器本地用于检测，不上传、不保存。
- 需要通过 `http://localhost:4173` 这类本地服务访问，直接打开文件时浏览器可能不允许摄像头权限。
- 人脸检测优先使用浏览器的 `FaceDetector` 能力；如果当前浏览器不支持，会自动切到本地 canvas 备用检测。
- 备用检测通过肤色区域估算人脸数量，精度不如原生检测，但不需要外部依赖。

## 本地预览

```bash
python3 -m http.server 4173
```

然后打开 `http://localhost:4173`。
