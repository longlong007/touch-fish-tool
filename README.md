# 摸鱼补给站

一个纯前端摸鱼小工具，直接打开 `index.html` 就能玩。

## 操作

- 拖动小鱼吃气泡，获得摸鱼值和连击。
- 大鲨鱼会持续追逐小鱼，被追上后本局结束。
- 撞到灰色障碍物会短暂减速，并清空连击。
- 吃到气泡可以把鲨鱼稍微甩开一点。
- 按 `Space` 暂停或继续。
- 按 `Esc` 快速隐藏，页面会切换成工作流看板。
- 按 `Alt + H` 也可以快速切换到伪装页面。
- 在工作流看板点击右上角返回按钮继续，游戏结束后可以重新开始。

## 本地预览

```bash
python3 -m http.server 4173
```

然后打开 `http://localhost:4173`。
